This repository contains a scaffold for the Nectar diet-planner web app (MERN).

Structure:

- server: Express + Mongoose backend (models, repository, services, controllers, routes, utils, config, middleware)
- client: React (Vite) frontend with folders: pages, components, hooks, context, assets

See server/ and client/ README sections for quick start.

Quick start

1. Install server dependencies:

```bash
cd server
npm install
```

2. Install client dependencies:

```bash
cd ../client
npm install
```

3. Run server (example):

```bash
cd server
npm run dev
```

4. Run client (example):

```bash
cd client
npm run dev
```
